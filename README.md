# BetterWix

BetterWix is a personal and professional website that showcases my skills, projects, portfolio, and personal profile. It serves as a digital portfolio for potential clients, collaborators, and friends. The website demonstrates both my professional expertise and personal interests.
https://wix-society.netlify.app/
---

## Features

- **Personal Introduction**: Presents my personality, character traits, and interests.
- **Professional Portfolio**: Displays my work as an artist, designer, web developer, audiobook maker, content creator, and data scientist.
- **Projects and Designs**: Visual showcase of projects, stories, and designs.
- **Skills Section**: Covers professional and personal skills such as Figma, Python, Adobe Suite, writing, photography, guitar, and more. Includes search functionality.
- **Education and Language Skills**: Lists education background and languages i speak (Italian, Russian, English, Persian).
- **Music Player**: Plays my favorite song, with adjustable controls for the user.
- **Portfolio Search and Filter**: Allows users to search and filter portfolio items efficiently.
- **Travel Map**: Shows visited locations and wishlist using Leaflet technology.
- **Responsive Design**: Website adapts to different screen sizes and devices.
- **Recommendations**: Displays testimonials and references from colleagues or clients.
- **Social Media Links**: Provides links to all social profiles for direct contact.
- **Animations**: Pure CSS and React.js animations for smooth visual effects.

---

## Technologies Used

- **Frontend**: React.js
- **Maps**: Leaflet
- **Translations**: i18n for multi-language support
- **Styling and Animations**: CSS and React.js
- **Other**: JavaScript, HTML, responsive design techniques

## How to Use

1. Open the website in a web browser. https://wix-society.netlify.app/
2. Explore portfolio projects and designs.
3. Browse the skills section and use search if needed.
4. Switch languages using the language panel.
5. Control music using the built-in music player.
6. View the travel map to see visited locations and wishlist places.
7. Contact via provided email or social media links.
